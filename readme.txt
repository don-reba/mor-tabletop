Добро пожаловать в корневую директорию сайта настолки по Мору. Как видите, сайт зависит от PHP; при этом он не требует базу данных.

Некоторые файлы нужны только для хостинга на Heroku — другим они ни к чему. Это: composer.json и Procfile. Директории .git и файлу readme.txt на сервере тоже не место. Необходимые файлы и директории это:
	- css/
	- en/
	- js/
	- ru/
	- index.php
	- subscribe.php
